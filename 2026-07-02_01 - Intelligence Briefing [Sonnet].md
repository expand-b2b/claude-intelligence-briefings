# Claude Intelligence Briefing - 2026-07-02

## Neue Releases

### Claude Code v2.1.197 (30. Juni 2026)
**Claude Sonnet 5 ist jetzt das Standardmodell in Claude Code.**

- Natürliches 1M-Token-Kontextfenster (kein kleineres Variant)
- Promotionspreise: $2/$10 pro Mtok bis 31. August 2026
- 128k Max Output Tokens, Adaptive Thinking
- Selektierbarer Reasoning-Aufwand: low, medium, high, max, x-high
- Leistung nahe an Opus 4.8, aber zu Sonnet-Preisen
- Substanzielle Verbesserungen bei Agentic-Performance, Tool Use, Coding

### Claude Code v2.1.196 (29. Juni 2026)
- Organisations-Standardmodelle konfigurierbar
- Lesbare Standardnamen für Sessions beim Start
- Klickbare Dateianhänge im Chat (Cmd/Ctrl-Klick öffnet Datei im Finder/Explorer)
- Fixes: Keyboard-Focus-Probleme bei Claude Agents Side Panel, Session-Status-Anzeige
- Verbesserte Stabilität von Background-Sessions (überleben Prozess-Neustarts)

Nach einer US-Regierungs-Exportkontroll-Sperre vom 12. Juni wurde der Zugang zu Fable 5 und Mythos 5 am 30. Juni wiederhergestellt. Seit 1. - Code Modernization Starter Kit für Legacy-Codebase-Migration
- Anthropic skaliert Partner-Team fünffach mit dediziertem Applied AI Engineering
- Große Beratungsfirmen an Bord: Accenture (30.000 Professionals), Cognizant (350.000), Deloitte (470.000), KPMG (276.000)

Neues Produkt für Forscher mit integrierten Tools und Paketen, auditierbare Artefakte, flexible Rechenressourcen. Verfügbar für Pro/Max/Team/Enterprise. Antragsphase für KI-Forschungsprojekte läuft bis 15. Juli 2026.
